Installation

Pour télécharger les clips depuis le site, tu as besoin de 3 outils sur ton PC Windows. Suis les étapes dans l'ordre — chaque commande se copie en un clic.

1
Python
Le langage qui fait tourner yt-dlp

Vérifie d'abord si Python est déjà installé :

python --version

Si tu vois Python 3.x.x → passe à l'étape 2. Sinon, télécharge Python ici :

↗ python.org/downloads

⚠ Pendant l'install, coche bien "Add Python to PATH" en bas de la fenêtre.

2
yt-dlp
L'outil qui télécharge les VODs Twitch

Ouvre un cmd (touches Win+R → tape cmd → Entrée) et colle ça :

pip install yt-dlp

Pour vérifier que c'est bien installé :

python -m yt_dlp --version
3
ffmpeg
Nécessaire pour découper les clips à la bonne durée

Toujours dans le cmd, colle ça (Windows 10/11) :

winget install ffmpeg

Ça télécharge ~230 MB et s'installe tout seul. Une fois fini, ferme le cmd et rouvre-en un nouveau, puis vérifie :

ffmpeg -version
✓
Tout est prêt !
Tu peux télécharger les clips depuis les cartes

Sur chaque tip, clique sur ⬇ clip — un fichier .bat se télécharge. Double-clique dessus, ça lance le téléchargement automatiquement dans C:\clips\.

Le clip fait ±5 min autour du timecode du tip (2 min avant, 3 min après).
